## SECTION 2 — Chronological Phases [UPSC-P] [UPSC-M]

```
IVC CHRONOLOGICAL PHASES
─────────────────────────────────────────────────────────────────
PHASE 1 → EARLY HARAPPAN (Pre-Harappan / Regionalisation)
           ~3300–2600 BCE
           • Village settlements → Proto-urban stage
           • Pottery + craft production begins
           • Sites: Mehrgarh (earliest base), Kot Diji, Amri, Hakra Ware
           • Kot Diji phase: precursor to mature Harappan culture
           ↓
PHASE 2 → MATURE HARAPPAN (Integration Phase) ★ MOST IMPORTANT
           ~2600–1900 BCE
           • Peak of urban civilisation
           • Standardised bricks, weights, script, town planning
           • All major cities (Mohenjo-Daro, Harappa, Lothal)
           • Long-distance trade with Mesopotamia
           ↓
PHASE 3 → LATE HARAPPAN (Localisation / Decline Phase)
           ~1900–1300 BCE
           • Urban centres decline
           • Regional cultures emerge: Cemetery H (Harappa), Jhukar (Sindh)
           • Loss of standard weights, script, town planning
           • Population shifts south & east
           ↓
POST-HARAPPAN
           • Painted Grey Ware (PGW) culture → Vedic age
           • Black & Red Ware → Megalithic South India
```

## SECTION 5 — Town Planning: In Depth [TNPSC] [UPSC-P]

### Two-Part City Structure

```
HARAPPAN CITY LAYOUT
════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────┐
│               CITADEL (Upper Town)                   │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐   │
│  │ GRANARY  │  │  GREAT   │  │  PUBLIC HALLS    │   │
│  │ (Stores) │  │  BATH    │  │  Assembly rooms  │   │
│  └──────────┘  └──────────┘  └──────────────────┘   │
│                                                      │
│  • Built on mud-brick platform (~6–12 m high)        │
│  • Administrative + religious center                 │
│  • Smaller population                                │
└────────────────────────┬────────────────────────────┘
                         │
┌────────────────────────┴────────────────────────────┐
│              LOWER TOWN (Residential)                │
│                                                      │
│   Main Streets (9–10 m wide) ──→ North–South        │
│         │                                            │
│         ├── Lanes (1.5–3 m wide) ──→ East–West      │
│         │         │                                  │
│         │         └── Houses (brick; courtyards)     │
│         │                                            │
│   Grid-iron / chess-board pattern                    │
│   Roads: right angles; flanked by houses            │
│   Each block: ~3–4 houses sharing walls             │
└─────────────────────────────────────────────────────┘

★ DHOLAVIRA exception: 3-part city (Citadel + Middle town + Lower town)
```

### Street & Road System

| Feature | Detail |
|---------|--------|
| Main streets | 9–10 m wide; N–S orientation |
| Cross lanes | 1.5–3 m wide; E–W |
| Pattern | Grid-iron (right angles) |
| Material | Mud; no paving in most cities |
| Lampposts? | Evidence of post holes at intervals (possibly lampposts or signs) |
| Dustbins | Evidence of rubbish jars at street corners |

### Drainage System — Most Advanced of Ancient World

```
DRAINAGE SYSTEM (Schematic)
─────────────────────────────────────────
HOUSE BATHROOM
    │
    ↓  (wastewater via chute in wall)
LANE DRAIN (brick-lined, covered with brick slabs)
    │
    ↓
MAIN STREET DRAIN (larger; deeper; covered)
    │
    ↓
SOAKAGE JAR / CESSPIT (for solid waste filtration)
    │
    ↓
OUTSIDE CITY (final disposal)

Key features:
• All drains: COVERED (inspection holes for cleaning)
• Material: BURNT bricks + lime plaster
• Houses had private bathrooms CONNECTED to public drains
• NO COMPARABLE system in ancient Mesopotamia or Egypt
```

### Houses — Structure Details

```
TYPICAL HARAPPAN HOUSE
    │
    ├── Entrance: from LANE (not main road) → Privacy
    ├── Central COURTYARD (open air)
    ├── Rooms arranged around courtyard
    ├── Bathroom: paved floor; brick walls; covered drain
    ├── Well: private wells in larger houses
    ├── Upper floors: evidence from staircase remains
    ├── Windows: NONE facing main street (only small ventilation slots)
    └── Material: BURNT bricks (ratio 1:2:4 → Height:Width:Length)
```

### Brick Specifications

| Feature | Detail |
|---------|--------|
| Standard ratio | **1 : 2 : 4** (Height : Width : Length) |
| Material (walls) | Unbaked/sun-dried mud bricks |
| Material (drains/bath) | **Burnt bricks** (kiln-fired) |
| Mortar | Mud mortar (gypsum used in Great Bath) |
| Size example | 28 × 14 × 7 cm |
| Uniformity | Same ratio maintained across all sites — thousands of km apart |

## SECTION 6 — Great Bath [UPSC-P] [TNPSC]

```
GREAT BATH — MOHENJO-DARO
═══════════════════════════════════════════════════════

Location: On the CITADEL (upper town)

Dimensions:
  → Length: 11.8 m (39 feet)
  → Width: 7 m (23 feet)
  → Depth: 2.4 m (8 feet)

Construction:
  → Double walls: outer mud-brick + inner burnt brick
  → Floor: burnt brick set in gypsum mortar
  → Watertight: bitumen (natural tar) sealant between layers
  → Steps on BOTH north and south ends (leading into water)

Surroundings:
  → Verandahs on sides
  → Small rooms (priests' changing rooms? — interpretation)
  → Large well nearby (water supply)
  → Drain: large corbelled drain (to empty/change water)

Purpose:
  → Ritual bathing (religious purification)
  → Public communal bath
  → Possible priestly use
  → Similar to later Hindu ritual bathing traditions

★ EXAM FACT: Great Bath is the EARLIEST known public water tank in the world
```

## SECTION 8 — Economy [TNPSC] [UPSC-P] [UPSC-M]

### Agriculture

```
HARAPPAN AGRICULTURE
──────────────────────────────────────────────────
CROPS GROWN:
  Cereals: Wheat (2 varieties), Barley (main crop), Rice (Lothal, Rangpur)
  Pulses: Peas, Lentils, Chickpea, Field beans
  Oil seeds: Sesame, Mustard
  Others: Cotton (FIRST IN WORLD to grow cotton), Dates, Melon

TECHNIQUES:
  → Ploughing: Kalibangan (world's first ploughed field; pre-Harappan layer)
  → Irrigation: Canal irrigation (evidence at Shortughai, Afghanistan)
  → Double cropping (winter + summer) — two crop seasons
  → Wells: private wells in cities (water access)

ANIMALS DOMESTICATED:
  → Zebu cattle (humped bull — seen on seals) ★
  → Sheep, Goat, Pig, Buffalo, Elephant, Dog, Cat, Fowl
  → HORSE: debated (possible evidence at Surkotada only)
  → NO confirmed horse use in IVC ← TNPSC hot fact

WILD ANIMALS HUNTED:
  → Tiger, Elephant, Deer, Bear (shown on seals and terracotta)
```

### Cotton — First Civilisation to Grow It

> **IVC = world's first civilisation to cultivate cotton**
> Greeks called cotton "Sindon" (from Sindhu/Indus)

### Trade

```
INTERNAL TRADE
──────────────────────────────────────────────────
Between IVC cities:
  → Standardised weights & measures ensured fair trade
  → Pottery, beads, metals, shell objects traded
  → Craft specialisation: each site produced specific goods
     • Nageshwar/Balakot → shell objects
     • Shortughai → lapis lazuli
     • Lothal → carnelian beads, copper tools
     • Chanhu-daro → bead-making, steatite work

EXTERNAL / LONG-DISTANCE TRADE
──────────────────────────────────────────────────
Trading partners:
  → Mesopotamia (Sumer, Akkad) — most important
  → Oman (Magan)
  → Bahrain (Dilmun)
  → Iran (Elam)
  → Central Asia (Shortughai — lapis lazuli source)
  → Afghanistan (lapis lazuli, tin)

Evidence:
  → Harappan seals found in Mesopotamia (Ur, Kish, Nippur)
  → Mesopotamian cylinder seals found in Harappan sites
  → Indus region = "MELUHHA" in cuneiform (Akkadian) texts
  → Dilmun = Bahrain; Magan = Oman (mentioned in Sumerian texts)

Commodities EXPORTED by IVC:
  → Cotton textiles, Carnelian beads, Lapis lazuli
  → Copper tools, Pottery, Timber
  → Pearls, Gold, Ivory

Commodities IMPORTED:
  → Tin (from Afghanistan/Central Asia)
  → Silver (from Iran/Afghanistan)
  → Lapis lazuli (Afghanistan — Shortughai)
  → Gold (from Karnataka/South India? — debated)
```

### Craft & Technology

```
HARAPPAN CRAFTS
──────────────────────────────────────────────────
METALLURGY:
  → Metals known: Copper, Bronze (copper + tin), Gold, Silver
  → IRON: NOT known ← important TNPSC/UPSC fact
  → Techniques: casting (open mould, lost-wax/cire perdue)
  → Dancing Girl = lost-wax casting technique
  → Copper tools: axes, saws, chisels, knives, fishhooks

POTTERY:
  → Wheel-made (potter's wheel used)
  → Red/orange ware with black painted designs (Black-on-Red ware)
  → Designs: geometrical, floral, fish-scale, pipal leaf, peacock
  → Perforated jars (colanders/strainers)
  → Miniature vessels

BEAD-MAKING:
  → Materials: Carnelian (red/orange), Lapis lazuli (blue), Steatite, Faience, Shell, Ivory, Gold
  → Lothal: carnelian bead factory
  → Technique: drilling holes with copper drill bits
  → Carnelian beads at Lothal: exported to Mesopotamia

SHELL INDUSTRY:
  → Nageshwar + Balakot: major shell-working centres
  → Shell bangles, ladles, inlays

WEAVING:
  → Cotton woven (spindle whorls found)
  → Impressions of woven cloth on pottery/seals

SEAL-MAKING:
  → Steatite seals: carved, fired, glazed white
  → Also: copper, faience, terracotta, ivory seals
```

## SECTION 9 — Weights & Measures [UPSC-P]

```
STANDARDISED WEIGHTS — BINARY SYSTEM
──────────────────────────────────────────────────
Ratio: 1 : 2 : 4 : 8 : 16 : 32 : 64 ...
         ↑
     Smallest weight = 13.63 grams (= 1 unit)

Material: Chert (cubical), Limestone, Sandstone
Shape: Cubic (lower values), Barrel/spherical (higher values)

MEASURES:
  → Scale/ruler: ivory + shell scale from Mohenjo-Daro
  → 1 Indus inch ≈ 1.75 cm
  → Lothal: ivory scale (smallest graduation = 1.77 mm)

SIGNIFICANCE:
  → SAME weights found across all IVC sites (uniform system)
  → Evidence of centralised authority OR strong mercantile standards
  → Mesopotamian merchants used IVC weights (found in Mesopotamia)
```

## SECTION 10 — Seals [TNPSC] [UPSC-P]

```
HARAPPAN SEALS — OVERVIEW
──────────────────────────────────────────────────
Total found: ~4,000+
Most common shape: Square with boss (projection) on back

MATERIALS:
  → Steatite (most common) — carved, fired, then glazed white
  → Copper, Faience, Terracotta, Ivory, Gold

CONTENT ON SEALS:
  → Animal depictions: unicorn (most common), humped bull (Zebu),
    elephant, rhinoceros, tiger, crocodile, buffalo
  → Human/deity figures: Pashupati seal, yogic figures
  → Inscriptions (4–5 signs usually; longest = 26 signs)

PURPOSE:
  → Commercial: identity/ownership mark on traded goods
  → Administrative: official identity
  → Religious amulets (some interpretations)
  → Impressions made on clay bullae (clay tags sealed on goods)
```

### Important Seals

| Seal | Description | Significance |
|------|-------------|-------------|
| **Pashupati Seal** | Three-faced figure seated in yoga posture; surrounded by 4 animals (elephant, tiger, rhinoceros, buffalo); 2 deer at feet | Proto-Shiva; earliest evidence of Shaivism/yoga |
| **Unicorn Seal** | Single-horned bull-like animal (most common seal motif) | Trade identity; possibly mythological |
| **Humped Bull (Zebu)** | Bull with prominent hump | Zebu cattle = sacred; fertility symbol |
| **Swastika Seal** | Swastika symbol | Good luck symbol (ancient; predates later uses) |
| **Persian Gulf Seal** | Round/circular seal (different from standard IVC square seal) | Found at Lothal; shows Gulf trade connection |
| **Mesopotamian Cylinder Seal** | Cylindrical seal (unlike Harappan flat) | Found at Harappan sites; confirms Mesopotamia trade |

> **UNICORN = Most depicted animal on Harappan seals** ← TNPSC hot fact

## SECTION 11 — Religion [UPSC-P] [TNPSC]

```
HARAPPAN RELIGION — KEY FEATURES
──────────────────────────────────────────────────
NO TEMPLES found ← very important fact
Evidence of NATURE WORSHIP:
  → Trees: Pipal tree (on seals); Neem
  → Animals: Zebu bull, Unicorn, Tiger, Crocodile
  → Water: Great Bath (ritual bathing)

MOTHER GODDESS:
  → Terracotta female figurines (thousands found)
  → Some with child; some with smoke or plants coming from womb
  → Fertility goddess interpretation
  → Widespread across all IVC sites

PASHUPATI (PROTO-SHIVA):
  → Seated on throne in yoga posture
  → Three faces (or possibly one face with headdress)
  → Surrounded by animals → "Lord of Animals" = Pashupati
  → Similar to later Shiva as Pashupati, Mahayogi, Maheshwara
  → Earliest evidence of SHIVA worship in India

FIRE WORSHIP:
  → Kalibangan: 7 fire altars on citadel (systematic fire ritual)
  → Lothal: fire altars found
  → Evidence: ash, charred animal bones, standardised pit structures

TREE/PLANT WORSHIP:
  → Pipal tree (Ficus religiosa) on seals
  → Deity standing inside/beneath pipal tree

BURIAL PRACTICES:
  → Extended inhumation (body laid flat; head toward north)
  → Grave goods: pottery, ornaments, tools
  → NO elaborate tombs or pyramids (unlike Egypt)
  → Cemetery H at Harappa (Late phase): fractional burial (body parts in urns — skull + limbs)
  → Lothal: double burial (man + woman buried together — rare)
  → Evidence of belief in afterlife (grave goods)

WHAT'S ABSENT:
  → No temples
  → No priest-king political authority confirmed
  → No war/conquest glorification
  → No royal tomb
```

## SECTION 13 — Script [UPSC-P] [UPSC-M]

```
HARAPPAN SCRIPT — KEY FACTS
──────────────────────────────────────────────────
Status: UNDECIPHERED (as of 2024)
Total inscriptions: ~5,000+
Direction: RIGHT to LEFT (boustrophedon style — alternating directions)
           Evidence: some longer texts reverse direction
Characters: ~400+ signs (logosyllabic? — not confirmed)
Where found: Seals, pottery, copper tablets, ivory rods, bronze tools
Longest text: 26 signs (on a seal)
Average length: 4–5 signs per seal

LANGUAGE THEORIES:
  → Proto-Dravidian: Asko Parpola (Finnish) — leading theory
  → Dravidian: Various Indian scholars
  → Proto-Sanskrit/Indo-Aryan: S.R. Rao (minority view)
  → Non-linguistic (just symbols): Steve Farmer et al. (controversial minority)

WHY UNDECIPHERED:
  → No bilingual inscription found (like Rosetta Stone)
  → Texts too short (most 4–5 signs)
  → No known related language to compare

SIGNIFICANCE OF LARGEST INSCRIPTION:
  → Dholavira: 10 wooden sign-boards (each ~37 cm × 25 cm)
  → Inscribed with 10 large signs (each ~37 cm tall)
  → Placed on city gate (like a name/proclamation board)
  → LARGEST HARAPPAN INSCRIPTION found so far
```

## SECTION 14 — Decline of IVC [UPSC-P] [UPSC-M]

```
DECLINE TIMELINE: ~1900 BCE — gradual collapse of mature phase

THEORIES OF DECLINE
══════════════════════════════════════════════════════════

1. ARYAN INVASION THEORY (Mortimer Wheeler) — ✗ DISCREDITED
   → Wheeler: skeletons at Mohenjo-Daro = massacre by invaders
   → Evidence: 37 skeletons in "disorderly" positions
   → REFUTED: skeletons are post-IVC deposits; no massacre evidence
   → Genetically and archaeologically rejected now

2. CLIMATE CHANGE & DRYING OF RIVERS (Most accepted)
   → Monsoon weakening ~2000–1700 BCE (paleoclimate data)
   → Ghaggar-Hakra (Saraswati) river dried up — maximum sites were on it
   → Saraswati drying = catastrophic for most IVC settlements
   → Gwen Robbins Schug: drought + disease combination

3. TECTONIC ACTIVITY / FLOODS
   → Mohenjo-Daro: evidence of repeated flooding
   → Tectonic uplift near Indus mouth → floods backed up
   → R.L. Raikes + Robert Dales: flooding theory for Mohenjo-Daro

4. EPIDEMIC / DISEASE
   → Skeletal evidence at Mohenjo-Daro shows signs of disease
   → Gwen Robbins Schug: inter-personal violence + disease in late phase

5. DECLINE OF TRADE
   → Mesopotamian trade records: Meluhha trade disappears ~1800 BCE
   → Loss of trade = economic collapse

6. OVERUSE OF RESOURCES / ECOLOGICAL DEGRADATION
   → Deforestation for brick kilns (millions of bricks needed fuel)
   → Soil degradation from intensive farming

WHAT HAPPENED TO PEOPLE?
─────────────────────────
→ EASTERN MIGRATION: shift to Gangetic plains (linked to later Vedic settlements)
→ SOUTHERN MIGRATION: toward Deccan and South India
→ Cemetery H culture at Harappa = different people or cultural change
→ No sudden disappearance — gradual decentralisation + rural life
```

## SECTION 15 — IVC & Vedic / Dravidian Connections [UPSC-M]

```
IVC ↔ VEDIC / DRAVIDIAN CONNECTION — DEBATE
═══════════════════════════════════════════════════════

ARGUMENT: IVC was DRAVIDIAN (Pro-Dravidian)
  → Script = possibly Proto-Dravidian (Asko Parpola)
  → Tamil-Brahmi script has some similarities to IVC signs
  → IVC deity (Pashupati) = ancestor of South Indian traditions
  → Absence of horse = Vedic culture was horse-centered; IVC was not
  → Pipal tree worship + Mother Goddess = Dravidian religious tradition

ARGUMENT: IVC was VEDIC / ARYAN (Pro-Vedic)
  → S.R. Rao deciphered script as Sanskrit (rejected by mainstream)
  → Kalibangan fire altars = Vedic yajnas?
  → IVC = Saraswati civilisation (Ghaggar-Hakra = ancient Saraswati)
  → N.S. Rajaram's "horse seal" = IVC had horses (rejected; altered photo)

GENETIC EVIDENCE (Recent: 2019 Rakhigarhi DNA study)
  → Rakhigarhi skeletons: NO steppe ancestry (no R1a gene = no Aryan gene)
  → IVC people = South Asian ancestry (ASI — Ancestral South Indians)
  → IVC people are ancestors of modern South Asians
  → Confirms IVC ≠ Vedic Aryan population

CONSENSUS VIEW (Current mainstream):
  → IVC people were indigenous South Asians
  → After IVC decline, Indo-Aryan speakers entered from Central Asia (~1500 BCE)
  → IVC traditions (Mother Goddess, Shiva, yoga) absorbed into later Hinduism
  → IVC is ancestor of Dravidian-speaking populations (likely)
```

## SECTION 17 — Dholavira: Special Focus [UPSC-P] [TNPSC]

```
DHOLAVIRA (Rann of Kutch, Gujarat)
════════════════════════════════════════════════════════

Excavation: R.S. Bisht (1989–2005)
Period: ~2900–1500 BCE (longest continuous IVC occupation)
Size: 100 hectares (one of largest IVC sites)
UNESCO: Inscribed as World Heritage Site in 2021

UNIQUE FEATURES:

1. THREE-PART CITY DIVISION (unique in IVC)
   → Citadel (upper) + Middle Town + Lower Town
   → Other cities: 2 parts (citadel + lower town)

2. WATER CONSERVATION SYSTEM (most sophisticated ancient water system)
   → 16 reservoirs covering 36 acres (36% of city area for water!)
   → Stepwells (forerunners of later Indian stepwells)
   → Channels from seasonal rivers → reservoir system
   → Dams to trap rainwater
   → Rock-cut reservoir (unique to Dholavira)

3. LARGEST HARAPPAN INSCRIPTION
   → 10 large wooden signboards at city gate
   → Each sign ~37 cm tall → visible from distance
   → 10 Indus script signs in a row
   → Like a city nameplate/proclamation board

4. STADIUM / SPORTS GROUND
   → Large open rectangular space between citadel and lower town
   → Seating on sides → interpreted as public assembly/stadium
   → First possible evidence of organised public events in India

5. UNIQUE GATE DESIGNS
   → Multiple gateways with different designs
   → Stone architecture (most IVC = brick; Dholavira uses stone too)
```

## SECTION 18 — Lothal: Special Focus [UPSC-P] [TNPSC]

```
LOTHAL (Gujarat — Sabarmati + Bhogavo rivers confluence)
════════════════════════════════════════════════════════

Excavation: S.R. Rao (1955–62), ASI

UNIQUE FEATURES:

1. DOCKYARD (World's earliest known)
   ─────────────────────────────────
   Size: 220 × 37 m (large trapezoidal basin)
   Depth: 4.15 m
   Connected to: Bhogavo river via channel
   Inlet: Sluice gates to control water level
   Purpose: Docking ships for loading/unloading cargo
   Significance: Shows maritime trade with Gulf/Mesopotamia

   ★ Note: Some scholars (S.P. Gupta) dispute it was a dockyard;
     may have been an irrigation reservoir → debate continues

2. FIRE ALTARS
   → Multiple fire altars in lower town
   → Most IVC sites: fire altars on CITADEL
   → Lothal: fire altars in LOWER TOWN (unusual)
   → May indicate fire ritual was more widespread/public here

3. BEAD FACTORY
   → Carnelian bead factory (mass production)
   → Copper tools for drilling
   → Finished beads exported to Mesopotamia (found in Ur/Sumer)

4. DOUBLE BURIAL
   → Man and woman buried together
   → Only example of double burial in IVC
   → May indicate belief in after-death companionship or sacrifice

5. PERSIAN GULF SEAL
   → Circular/button seal (different from standard IVC square seal)
   → Found at Lothal → indicates Gulf traders visited / lived here

6. RICE HUSK
   → Evidence of rice cultivation near Lothal
   → One of earliest evidence of rice in Indian subcontinent
```

## SECTION 19 — Kalibangan: Special Focus [UPSC-P]

```
KALIBANGAN (Rajasthan — Ghaggar river)
════════════════════════════════════════════════════════

Excavation: B.B. Lal + B.K. Thapar (1960–69)
Name meaning: "Black bangles" (kali = black; bangan = bangles)

TWO PHASES:
  Phase I (Pre-Harappan): ~2920–2550 BCE
  Phase II (Mature Harappan): ~2550–1800 BCE

UNIQUE FEATURES:

1. WORLD'S FIRST PLOUGHED FIELD (Phase I = Pre-Harappan!)
   → Grid of furrows (plough lines) preserved in clay
   → Two sets crossing each other at right angles
   → Modern Punjab still uses same pattern for double-crop fields
   → Significance: earliest direct evidence of ploughing ANYWHERE IN THE WORLD

2. FIRE ALTARS (Phase II Mature Harappan)
   → 7 fire altars in a row on the CITADEL
   → Standardised pits; ash; animal bones; clay figurines found
   → Evidence of fire ritual (not proven to be Vedic yajna)
   → Also: fire altars in LOWER TOWN (domestic fire ritual)

3. EARTHQUAKE EVIDENCE
   → Phase I destruction layer: evidence of earthquake (~2600 BCE)
   → Broken walls; distorted brick courses
   → Only IVC site with earthquake evidence in pre-Harappan layer

4. CAMEL BONES
   → Only IVC site to yield camel bones
   → Suggests camel was known (though not domesticated for transport?)

5. NO TOILET / BATH IN EVERY HOUSE
   → Unlike Mohenjo-Daro/Harappa
   → Only some houses had baths → less standardised sanitation
```
