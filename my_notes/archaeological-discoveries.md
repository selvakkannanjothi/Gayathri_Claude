## SECTION 2 — Pre-Historic India: Chronological Timeline [TNPSC] [UPSC-P]

```
PRE-HISTORIC INDIA TIMELINE
─────────────────────────────────────────────────────────
2,000,000 BCE →  LOWER PALAEOLITHIC
                  • Homo erectus
                  • Tools: Hand axes, cleavers, choppers
                  • Key sites: Athirampakkam, Pallavaram, Bhimbetka
                  • First found in India: Pallavaram, 1863 (Robert Bruce Foote)
                  ↓
4,00,000 BP  →   MIDDLE PALAEOLITHIC
                  • Homo erectus → Archaic Homo sapiens emerging
                  • Smaller tools; scrapers, knives, Levallois flakes
                  • Identified first by H.D. Sankalia at Nevasa (Pravara river)
                  ↓
~30,000 BCE  →   UPPER PALAEOLITHIC
                  • Anatomically modern Homo sapiens
                  • Blade tools, burins
                  ↓
~8,000 BCE   →   MESOLITHIC (Middle Stone Age)
                  • Microliths (tiny stone tools)
                  • Hunting + gathering + fishing
                  • Key sites: Baghor (MP), Sarai Nahar Rai (UP)
                  ↓
~2,000 BCE   →   NEOLITHIC (New Stone Age)
                  • Polished stone axes (celts)
                  • Crop cultivation + animal domestication
                  • Key sites: Mehrgarh (Pakistan), Paiyampalli (TN)
                  ↓
~1,300 BCE   →   CHALCOLITHIC (Copper-Stone Age)
                  • Copper + stone tools
                  • Rural, pastoral communities
                  • Key sites: Inamgaon (Maharashtra), Jorwe (Maharashtra)
                  ↓
~1,000 BCE   →   IRON AGE / MEGALITHIC
                  • Iron technology
                  • Megalithic burial mounds
                  • Key sites (TN): Adichanallur, Kodumanal, Paiyampalli
                  ↓
~300 BCE     →   EARLY HISTORIC (Sangam Age in South India)
```

| Tradition | Tools | Region | Named After |

## SECTION 4 — Indus Valley Civilisation (IVC / Harappan Civilisation) [TNPSC] [UPSC-P] [UPSC-M]

### Quick Snapshot

| | Detail |
|---|---|
| Period | ~3000–1300 BCE; **Mature phase: 2600–1900 BCE** |
| Age | **Bronze Age** (no iron; copper + bronze used) |
| Geographical area | ~1.5 million sq. km |
| Core area | Pakistan, Gujarat, Rajasthan, Haryana |
| First discovered | **Harappa** by **Daya Ram Sahni** (1921) |
| Second site | **Mohenjo-Daro** by **R.D. Banerji** (1922) |
| Key director | **Sir John Marshall** (Director General of ASI — initiated systematic research) |
| Later excavator | **Mortimer Wheeler** (1940s) |
| Script | Undeciphered; ~5,000 texts; longest text = 26 signs |

### Boundaries of IVC

```
North:   Shortughai (Afghanistan)
West:    Sutkagen-dor (Pakistan–Iran border)
East:    Alamgirpur (Uttar Pradesh, India)
South:   Daimabad (Maharashtra, India)
```

## SECTION 6 — IVC: Town Planning & Features [TNPSC] [UPSC-P]

### Town Layout

```
HARAPPAN TOWN PLAN
    │
    ├── CITADEL (Upper town)
    │   • Public buildings, granary, Great Bath
    │   • Built on raised platform
    │   • Administrative/religious functions
    │
    └── LOWER TOWN
        • Residential area
        • Grid-pattern streets (roads at right angles)
        • Houses: multi-room, courtyard, upper floors
        • Bathrooms: burnt brick; connected to drains
        • Drainage: covered drains; burnt bricks
        • Wells in houses
```

### IVC Key Features

| Feature | Detail |
|---------|--------|
| **Bricks** | Standardized ratio — 1:2:4 (H:W:L); burnt bricks for drains, unbaked for houses |
| **Drainage** | Underground covered drainage — most advanced of ancient world |
| **Great Bath** | Mohenjo-Daro; courtyard with tank; watertight (gypsum mortar); ritual bathing |
| **Script** | Not deciphered; possibly Dravidian; ~5000 texts; inscribed on seals |
| **Seals** | Made of steatite, copper, terracotta, ivory; used as identity/ownership markers |
| **Weights** | Cubical chert; binary system (1:2:4:8:16:32); smallest unit = 13.63 grams |
| **Measurement** | Scale: 1 inch ≈ 1.75 cm |
| **Trade with Mesopotamia** | Harappan seals found in Oman, Bahrain, Iraq, Iran; "Meluhha" = Indus in cuneiform texts |
| **Crops** | Wheat, barley, lentil, chickpea, sesame, millets; double cropping |
| **Animals** | Sheep, goat, fowl, buffalo, pig, elephant; **HORSE NOT KNOWN** |
| **Harappan cattle** | Zebu (large breed; shown on seals) |

### Famous Artefacts

```
DANCING GIRL (Mohenjo-Daro)
→ Bronze statue; 10.5 cm tall
→ One hand on hip; bangles on arms up to upper arm
→ Made by LOST-WAX CASTING technique
→ Shows advanced metallurgical skills

PRIEST KING (Mohenjo-Daro)
→ Steatite (soapstone) sculpture
→ Wearing trefoil-patterned shawl
→ Indicates priestly/royal authority

PASHUPATI SEAL
→ Figure seated in yogic posture
→ Surrounded by animals (elephant, tiger, rhinoceros, buffalo)
→ Associated with proto-Shiva worship
```

## SECTION 7 — Decline of IVC [UPSC-P] [UPSC-M]

```
DECLINE OF HARAPPAN CIVILISATION (~1900 BCE)

Reasons given by historians:
    ├── Climate change (rainfall decrease)
    ├── Drying of rivers and water resources (continuous drought)
    ├── Decline of trade with Mesopotamia
    ├── Floods (especially at Mohenjo-Daro)
    ├── Shifting of river course (Ghaggar-Hakra / Saraswati dried up)
    └── Invasions (Aryan invasion theory — largely debated/discredited)

Post-decline:
→ People shifted SOUTH and EAST from Indus region
→ Second urbanisation: Sangam-era towns (Arikamedu, Keezhadi, Uraiyur)
   ← different from Indus cities
→ IVC → Tamil connection: Graffiti on megalithic burial pots of South India
   similar to Indus symbols; debated but not conclusively proven
```

## SECTION 9 — Megalithic/Iron Age Sites [TNPSC] [UPSC-P]

### What is a Megalith?

```
MEGALITH = Large stone monument used for burials
    │
    ├── TYPES:
    │   ├── Dolmen — flat stone slab on vertical stones (table-like)
    │   ├── Menhir — single upright tall stone; memorial
    │   ├── Cist — stone-lined burial chamber (primary/secondary)
    │   ├── Urn burial — burial in large pottery urns
    │   ├── Cairn circle — circular ring of stones over burial
    │   └── Rock-cut chamber — chambers cut into rock
    │
    └── SIGNIFICANCE:
        → Iron Age + Megalithic period = contemporary in South India
        → Black & Red Ware pottery = characteristic
        → Grave goods: iron weapons, beads, pottery, bronze objects
        → Belief in afterlife (grave goods placed with dead)
```

### Major Megalithic Sites of India

| Site | State | Key Finding |
|------|-------|------------|
| **Adichanallur** | Tamil Nadu (Thoothukudi) | Urn burials; iron weapons; rice grains; gold ornaments; excavated 1876 by Andrew Jagor (German); Alexander Rea (ASI 1902-03) |
| **Kodumanal** | Tamil Nadu (Erode) | Carnelian beads; iron swords (169 cm!); Brahmi inscription; iron smelting; on Noyyal river (Cauvery tributary) |
| **Paiyampalli** | Tamil Nadu (Tirupathur) | Urn burials; black & red ware; 1000 BCE; ASI excavated 1960s |
| **Brahmagiri** | Karnataka | First systematic megalithic excavation (Mortimer Wheeler) |
| **Sanur** | Tamil Nadu (Madhuranthakam) | Megalithic burials |
| **Sithannavasal** | Tamil Nadu (Pudukkottai) | Iron Age; Jain cave paintings (later period) |
| **Porunthal** | Tamil Nadu | Rice grains in burial urns (like Adichanallur) |
| **Nagarjunakonda** | Andhra Pradesh | Buddhist stupas; megalithic remains |

### Adichanallur — Detailed [TNPSC HIGH PRIORITY]

```
ADICHANALLUR (Thoothukudi district; 22 km from Tirunelveli)

Excavation History:
→ 1876: Andrew Jagor (German naturalist) — first excavator
→ Then: A.J. Stuart (District Collector) + Bishop Robert Caldwell
   → Found quartz site; quarrying banned
→ Alexander Rea: Comprehensive excavation; published in ASI Annual Report 1902-03
→ Later: ASI conducted another excavation

Findings:
→ Urns and pottery (large numbers)
→ Iron implements: spades, daggers, swords, spears, arrows
→ Bronze objects: buffalo, goat/sheep, cock, tiger, antelope, elephant figures
→ Stone beads + gold ornaments
→ Traces of cloth and wood
→ Rice husks and millet → evidence of grain domestication
→ Sacrificial implements → Caldwell concluded NOT Vedic religion followers

Significance:
→ Confirms Iron Age / pre-Sangam culture in Tamil Nadu
→ Iron weapons = advanced metallurgy
→ Rice + millet = agriculture
→ NOT influenced by Vedic traditions
```

### Kodumanal — Detailed [TNPSC]

```
KODUMANAL (40 km from Erode; Noyyal river, tributary of Cauvery)
→ Excavations: 1980s, 1990s, 2012
→ Two cultural periods: Megalithic + Early Historic
→ Carnelian beads — IDENTICAL to Mohenjo-Daro beads
   → Carnelian not local → IMPORTED from outside (trade connection with North India)
→ Iron sword: 169 cm long
→ Brahmi writing (similar to Ashokan Brahmi)
→ Iron smelting evidence
→ 782 carnelian beads found in single urn
→ Copper toddy filter with lotus and peacock designs
```

### Keezhadi — TNPSC HIGH PRIORITY

```
KEEZHADI (Sivaganga district; on Vaigai river)
→ Excavation started: 2015 (Tamil Nadu Archaeology Department)
→ Period: ~300 BCE (overlaps early Sangam Age)

Major Findings:
→ Brick structures: well-planned urban settlement
→ Tamil-Brahmi inscribed potsherds (earliest urban Tamil writing)
→ Shell bangles (matching site: Keezhadi)
→ Pearl (found at excavation)
→ Spindle whorls (weaving activity)
→ Copper objects, iron tools
→ Animal bones, charcoal
→ Ring wells

Significance:
→ Proves 2nd urbanisation in South India was contemporary with Sangam period
→ Challenges idea that Sangam society was purely rural
→ Tamil Brahmi inscription on pottery = literacy in Sangam society
→ Link to Sangam literature confirmed by evidence
```
